# Privacy Policy for MoneyBot Last Updated: May 12, 2026 1. Introduction MoneyBot ("we," "our," or "the App") is a personal automation tool designed to assist the owner in managing and uploading video content to their personal social media accounts. This Privacy Policy explains how we handle data. 2. Data Collection Personal Data: MoneyBot does not collect, store, or share any personal identification information (PII) on external servers. Media Files: The App processes video files stored locally on the user's hardware. These files are never uploaded to our servers; they are sent directly to the official APIs of TikTok, YouTube, and Instagram. Authentication: The App uses official OAuth 2.0 protocols to interact with social media platforms. Your login credentials (passwords) are never seen or stored by the App. 3. Third-Party Services MoneyBot interacts with the following third-party APIs: TikTok Content Posting API YouTube Data API v3 Instagram Graph API Please refer to their respective privacy policies regarding how they handle data once uploaded. 4. Data Security All processing happens locally on the user's machine. We implement industry-standard practices to ensure that API tokens are stored securely within the local environment. 5. Contact For any questions regarding this personal project, please contact the developer via the official GitHub repository or Telegram bot interface.
